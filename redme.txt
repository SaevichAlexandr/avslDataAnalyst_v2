Инструкция запуска приложения:

1. Запустить selenium через командную строку. Для этого вбить следующую команду:
java -Dwebdriver.gecko.driver="geckodriver.exe" -jar selenium-server-standalone-3.141.59.jar (версию видно на файле)

2. Далее необходимо запустить symfony server. Для этого в новой командной строке вбиваем:
symfony server:start

Важно: терминалы закрывать нельзя, иначе процессы закроются, потому для манипуляций с гитом и прочим нужно открывать
новый терминал.

3. Далее все запросы можно отсылать через postman. Сейчас они все находятся в папке Diplom.
